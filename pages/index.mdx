# Guide to Codex
 
This is a guide to Codex, a **simple, powerful AI translation editor**.

## Controls

| Icon                           | Description                    | Learn More                  |
|--------------------------------|--------------------------------|-----------------------------|
| <GenerateButtonSVG /> | Translates text from one language to another. | [Translate](https://guide.codex.ai/getting-started#translation) |
| <ResetButtonSVG /> | Resets the current page to a clean state. | [Reset](https://guide.codex.ai/getting-started#resetting) |
| <PromptButtonSVG /> | Opens the prompt terminal to edit text with natural language. | [Prompt Terminal](https://guide.codex.ai/tools/prompt_terminal) |
| <AudioButtonSVG /> | Reads the content or translation aloud. | [Audio](https://guide.codex.ai/tools/audio) |
| <BookButtonSVG /> | Opens the interface to add, delete, or navigate through pages within a translation. | [Pages](https://guide.codex.ai/pages) |
| <LayerButtonSVG /> | Splits content that is too long into multiple pages. | [Paginate](https://guide.codex.ai/pages#paginate) |
